### Зависимости
```bash
npm i -g webpack || yarn add global webpack
npm i			 || yarn
```

### Сборка
Девелоп сборка c `browser-sync`:
```bash
npm run build:dev  || yarn build:dev
```
Продакшн сборка:
```bash
npm run build:prod || yarn build:prod
```
